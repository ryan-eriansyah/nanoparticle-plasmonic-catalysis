# Core Size Dependence in Pulsed Laser-Induced Plasmonic Catalysis Observed in Gold@silver Core@shell Nanoparticles Synthesis

## Overview
This repository contains supporting materials for my undergraduate Physics thesis at Universitas Indonesia.

## Research Objectives
- Investigate the effect of gold core size on the silver shell growth rate under pulsed laser irradiaton.

## Key Findings
- Silver shell growth increases with laser power above a certain threshold.
- Plasmonic catalytic activity depends on the resonance condition between the laser wavelength and nanoparticle LSPR.
- Gold nanoparticle core size significantly affects the growth dynamics of the silver shell.
